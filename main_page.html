1.Introduction to Links

Hyperlinks are clickable elements that allow users to navigate between pages of the same website, or link to a different website.

1.1.The <a> tag & The href Attribute 1.2.Hyperlink states

  • This tag is used for links and primarily relies on the href attribute.

  • The href attribute can be an absolute URL or a relative URL.

  • To add a clickable link text, add it between the opening and closing tab.

Example: <a href="https://www.example.com/">Visit our website</a>

The states enhance user navigation:

  • Unvisited links are underlined in blue.
  • Active links, when clicked on, are underlined in red.
  • Visited links are underlined in purple.
